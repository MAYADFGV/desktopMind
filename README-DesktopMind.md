# DesktopMind 使用说明

## 这是什么

DesktopMind 是一个运行在 Windows 桌面上的透明思维工作台。

你可以把它理解成一层覆盖在桌面上的可编辑画布：

- 可以创建便利贴、文本、待办、文件卡片、图片、书签
- 可以给节点之间建立连线
- 可以分组、搜索、打标签
- 可以一边整理思路，一边保留原来的桌面和窗口工作流

它适合用来：

- 整理项目结构
- 记录临时灵感
- 做任务拆解
- 搭建资料关联图
- 在桌面上长期摆放自己的工作上下文

## 启动方式

### 方式一：直接双击启动脚本

在项目根目录中双击：

- `start-desktopmind.bat`

如果你使用中文启动脚本，也可以双击：

- `启动DesktopMind.bat`

### 方式二：命令行启动

在项目根目录打开终端后执行：

```bash
npm run tauri dev
```

如果你还没有安装依赖，可以先执行：

```bash
npm install
```

如果你使用的是 pnpm：

```bash
pnpm install
pnpm tauri dev
```

## 第一次启动前需要什么

你需要先准备好下面这些环境：

- Node.js 18 或更高版本
- Rust
- Windows 10 / 11

如果启动时报错，优先检查：

- Node.js 是否安装成功
- Rust / Cargo 是否在环境变量里
- 当前项目依赖是否已安装

## DesktopMind 的两种模式

DesktopMind 目前采用 **编辑模式 / 穿透模式** 双模式。

这是为了避免桌面覆盖层和系统窗口发生严重交互冲突。

### 1. 编辑模式

在编辑模式下，你可以正常操作 DesktopMind：

- 点击节点
- 拖动节点
- 编辑内容
- 连线
- 搜索
- 右键菜单
- 框选
- 标签和分组操作

### 2. 穿透模式

在穿透模式下，DesktopMind 会放行鼠标事件，方便你操作桌面或其他窗口。

这时你可以：

- 点击桌面图标
- 使用底层应用窗口
- 正常和其他软件交互

但注意：

**穿透模式下，DesktopMind 自己的按钮和节点也无法点击。**

这是当前方案的正常行为，不是 bug。

### 模式切换快捷键

- `Ctrl + Shift + P`：切换编辑 / 穿透模式
- `Ctrl + Shift + E`：强制回到编辑模式
- `Ctrl + Shift + Space`：显示 / 隐藏整个覆盖层

推荐用法：

- 需要整理内容时，用编辑模式
- 需要点桌面、操作别的窗口时，切到穿透模式
- 想继续编辑时，按 `Ctrl + Shift + E`

## 基础使用流程

### 1. 创建节点

你可以通过以下方式创建内容：

#### 鼠标方式

- 双击画布空白处：新建便利贴
- 点击工具栏按钮：新建不同类型节点
- 右键画布空白处：新建节点 / 粘贴内容

#### 键盘方式

- `N`：新建便利贴
- `T`：新建文本片段
- `Shift + T`：新建待办清单
- `B`：新建网页书签

新建后通常会直接进入编辑状态。

---

### 2. 编辑节点

不同类型节点支持不同编辑方式：

- 便利贴：标题 + 正文
- 文本片段：标题 + 文本内容
- 待办节点：标题 + 多条待办项
- 书签节点：网址和书签信息
- 分组节点：分组标题

常用快捷键：

- `Enter`：编辑当前选中节点
- `F2`：重命名当前选中节点
- `Ctrl + Enter`：结束当前编辑（部分节点可用）

---

### 3. 拖动和缩放

#### 画布操作

- 滚轮：缩放画布
- 中键拖拽：平移画布
- 右键拖拽空白区域：平移画布
- `Space + 左键拖拽`：平移画布
- `Ctrl + 0`：重置缩放

#### 节点操作

- 左键拖动节点：移动节点
- 拖节点边缘 / 角：调整大小

---

### 4. 建立连线

节点四周会显示连接锚点。

操作方式：

1. 将鼠标移动到节点边缘锚点
2. 按住拖出连线
3. 连接到另一个节点的锚点

建立连线后，你可以：

- 设置连线文字
- 调整颜色
- 调整线型
- 设置箭头方向
- 删除连线

---

### 5. 文件、图片和书签

#### 文件卡片

- 直接把文件从资源管理器拖到画布
- 会自动生成文件卡片
- 双击文件卡片可用系统默认程序打开

#### 图片节点

- 可以从剪贴板粘贴图片
- 图片文件也会生成带预览的节点或文件卡片

#### 网页书签

- 按 `B` 新建书签
- 或粘贴网址
- 程序会尝试抓取网页标题、图标和描述

---

## 选择、多选与批量操作

### 选择节点

- 单击节点：选中
- 拖动空白处框选：多选
- `Ctrl + A`：选择当前可见节点

### 常用批量操作

- `Delete`：删除选中内容
- `Ctrl + C`：复制选中节点
- `Ctrl + V`：粘贴
- `Ctrl + D`：复制并生成副本
- `Ctrl + G`：将选中节点创建为分组
- `Ctrl + Shift + G`：解除分组

---

## 标签和分组

### 标签

你可以给节点添加标签，用于后续筛选和搜索。

使用方式：

- 右键节点，选择编辑标签
- 或通过工具栏中的标签入口批量设置

标签适合用来标记：

- 项目阶段
- 优先级
- 模块归属
- 个人分类体系

### 分组

分组适合把一批相关节点组织成一个可折叠区域。

你可以：

- 将多个节点创建为分组
- 修改分组标题
- 改分组颜色
- 折叠 / 展开分组
- 添加节点到已有分组
- 从分组移除节点

---

## 搜索与快捷操作

### 搜索

按：

- `Ctrl + F`

可以打开搜索面板。

搜索支持：

- 关键词搜索
- 按节点类型筛选
- 按标签筛选
- 点击结果后聚焦到对应节点

### 快捷操作面板

按：

- `Ctrl + K`

可以打开快捷操作面板。

你可以在里面快速执行常见动作，例如：

- 新建便利贴
- 新建文本
- 新建待办
- 新建书签
- 打开搜索
- 创建分组
- 切换穿透模式
- 切换主题
- 重置视图

### 快捷键帮助

按：

- `?`

可以打开快捷键帮助面板。

---

## Markdown 支持

便利贴和文本片段支持基础 Markdown 预览。

支持的常见内容包括：

- 标题
- 列表
- 加粗 / 斜体
- 引用
- 行内代码
- 代码块
- 链接
- 任务列表

通常可以：

- 双击进入编辑
- 非编辑状态查看渲染后的预览

---

## 自动保存

DesktopMind 会自动保存当前画布。

当前逻辑包括：

- 编辑后自动防抖保存
- 隐藏前保存
- 失焦前保存
- 关闭前尝试保存

这意味着一般情况下，你不需要手动点击保存。

但在开发版本中，仍然建议：

- 重要修改后停留几秒
- 退出前确认终端没有报错

## 推荐使用方式

如果你是第一次使用，推荐按这个顺序熟悉：

1. 启动软件，保持在编辑模式
2. 双击空白处创建几个便利贴
3. 试着拖动、缩放和连线
4. 按 `Ctrl + F` 测试搜索
5. 给节点加标签
6. 选中多个节点后按 `Ctrl + G` 创建分组
7. 试一次 `Ctrl + Shift + P` 切换到穿透模式
8. 操作完桌面后按 `Ctrl + Shift + E` 返回编辑模式

这样会最快理解这套软件的工作方式。

## 常见问题

### 1. 为什么我切到穿透模式后点不了软件本身？

因为穿透模式会让 DesktopMind 放行鼠标事件到底层桌面或窗口。

这是设计行为。

返回编辑模式请按：

- `Ctrl + Shift + E`

或：

- `Ctrl + Shift + P`

### 2. 为什么它和普通窗口软件不一样？

因为 DesktopMind 不是传统主窗口工具，而是“贴在桌面上的覆盖层工作台”。

这让它更适合长期摆放信息，但也意味着它必须处理和桌面/其他窗口之间的交互关系，所以采用了编辑/穿透双模式。

### 3. 文件卡片显示失效怎么办？

如果原文件被移动、删除或路径变化，文件卡片会显示失效状态。

你可以：

- 检查原文件是否仍存在
- 重新拖入文件
- 删除失效节点并重新创建

### 4. 书签抓取失败怎么办？

有些网页会限制抓取标题、图标或描述。

这时书签节点仍然可以保留网址本身，只是元数据可能不完整。
